## PO/Gettext rules

- Do not translate `msgid`.
- Only translate `msgstr` for the target locale.
- For Vietnamese locale files, `msgstr` must be Vietnamese.
- Keep source references like `#: src/...` unchanged.
- Keep placeholders unchanged, for example `%s`, `{name}`, `{{count}}`.
- Do not translate route paths, component names, enum values, API fields, or variable names.
