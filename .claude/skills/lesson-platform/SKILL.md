# Lesson Platform Skill

Use this skill for:

- lesson workflow
- exercise builder
- student submissions
- progress tracking
- teacher dashboard
- role-based learning flows

## Business rules

- STUDENT can:
  - view lessons
  - submit answers
  - view progress

- TEACHER can:
  - create/edit/publish lessons
  - create exercises
  - review submissions
  - track student progress

- ADMIN can:
  - manage all users/classes/courses

## UI rules

- Keep public landing navigation unchanged.
- Role-based menus apply only inside dashboard/manage routes.
- Keep modern SaaS UI consistency.
- Keep localization support.
- Use reusable cards/forms/tables.
